PikaBot, a very good multi-purpose bot easy to use!

What is PikaBot? It's a bot that has multiples features like moderation features, fun commands and more! To get a list of commands, just type pika help

This bot also offers tags, and here is how they work:
First, You create a tag with the tag-create command like this: pika tag-create sleep Please sleep more!

Second, you use the tag with pika sleep and the bot will say "Please sleep more"! As easy as that! The tag-delete and tag-list commands are self-explanaory I guess.


Tags are one of the many features PikaBot have, so dont hesitate and try out everything!
